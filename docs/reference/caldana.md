# Arabidopsis thaliana growth experiment

A dataset containing 67 metabolites from plants grown under different
light and temperature conditions. This subset of the data contains only
the light effect and time effect for limited conditions, while the full
data also contains gene expressions.

## Usage

``` r
data(caldana)
```

## Format

A data.frame having 140 rows and 3 variables:

- light:

  Light levels

- time:

  Time of measurement

- compound:

  Metabolic compounds

## References

Caldana C, Degenkolbe T, Cuadros-Inostroza A, Klie S, Sulpice R, Leisse
A, et al. High-density kinetic analysis of the metabolomic and
transcriptomic response of Arabidopsis to eight environmental
conditions. Plant J. 2011;67(5):869-884.
